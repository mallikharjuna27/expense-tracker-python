# Expense Tracker Application

## 📌 Overview
A command-line based Expense Tracker built using Python to record, manage, and analyze daily expenses efficiently.

## 🚀 Features
- Add new expenses with date, category, amount, and description  
- View all recorded expenses in a structured format  
- Calculate total expenditure  
- Simple and user-friendly menu-driven interface  

## 🛠️ Tech Stack
- Python  
- File Handling
